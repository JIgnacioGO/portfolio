Sexto Proyecto - Simulador Farmacéutico

Desarrollé una aplicación web para simular un sistema de gestión farmacéutica utilizando Django, HTML, CSS, Bootstrap y PostgreSQL. La aplicación cuenta con un sistema de registro e inicio de sesión. Una vez autenticados, los usuarios pueden registrar nuevos laboratorios, directores de laboratorio y productos farmacéuticos. Se implementaron operaciones CRUD (Crear, Leer, Actualizar, Eliminar) para gestionar esta información almacenada en una base de datos PostgreSQL. El proyecto me permitió profundizar en el desarrollo de aplicaciones web con Django, enrutamiento, vistas basadas en clases, plantillas HTML y manipulación de bases de datos relacionales con PostgreSQL, incluyendo relaciones uno a muchos entre las entidades del sistema.
