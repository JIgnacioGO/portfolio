Quinto Proyecto - Catálogo de Autos

Desarrollé una aplicación web básica para gestionar un catálogo de autos, utilizando HTML, CSS, Bootstrap, Django y SQLite. La aplicación cuenta con un sistema de registro e inicio de sesión para que los usuarios puedan ingresar. Una vez logueados, los usuarios con permisos pueden agregar nuevos autos al catálogo, que se almacenan en una base de datos SQLite. Todos los usuarios registrados pueden ver el listado completo de autos. Adicionalmente, el superusuario tiene permiso para eliminar autos del catálogo. Este proyecto me permitió practicar el desarrollo de aplicaciones web básicas con Django, creación de plantillas HTML, enrutamiento, vista basadas en clases, y manipulación de bases de datos sencillas con SQLite.
