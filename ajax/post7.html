Septimo Proyecto - Proyecto final de Bootcamp, Aplicación de Gestión de Datos

Como proyecto final del bootcamp desarrollé una aplicación web para gestionar y filtrar datos utilizando HTML, CSS, Bootstrap, Django y PostgreSQL. La aplicación cuenta con registro e inicio de sesión de usuarios. Una vez autenticados, se puede visualizar información filtrada dinámicamente de una base de datos PostgreSQL provista por el bootcamp. Implementé varias vistas con diferentes tipos de filtros sobre los datos para cumplir con los requerimientos solicitados. El proyecto me permitió poner en práctica todos los conceptos de desarrollo web aprendidos durante el bootcamp, incluyendo planeación de una aplicación, vistas basadas en clases, procesamiento de formularios, consultas dinámicas a bases de datos, diseño responsive con Bootstrap, entre otros. Fue un proyecto muy desafiante pero me ayudó a consolidar mis habilidades de desarrollo fullstack.
