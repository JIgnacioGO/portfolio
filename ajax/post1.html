Primer Proyecto - Frutapp

Desarrollé una sencilla aplicación web llamada Frutapp que permite realizar pedidos de paletas. La interfaz fue creada con HTML y CSS básicos. Cuando el usuario hace clic en el botón "Haga su pedido aquí", se abre una ventana modal utilizando JavaScript, donde el usuario puede ingresar la cantidad de paletas que desea ordenar. Luego de ingresar la cantidad y confirmar el pedido, la ventana modal se cierra sin mostrar ningún mensaje confirmando la recepción del pedido. Este pequeño proyecto me permitió poner en práctica conceptos básicos de HTML, CSS y lógica en JavaScript, manejar eventos y manipular el DOM para mostrar u ocultar elementos
