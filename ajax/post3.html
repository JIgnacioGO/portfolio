Tercer Proyecto - Portafolio usando Bootstrap

Desarrollé un sitio web personal básico utilizando HTML, CSS, JavaScript y Bootstrap para crear un portafolio online. La página contiene una breve introducción sobre mí, así como una sección para mostrar proyectos, habilidades y contacto. Usé componentes de Bootstrap como navbar, cards y formularios para agilizar el desarrollo y lograr un diseño responsive. Este proyecto me permitió practicar la maquetación de sitios web con Bootstrap y mejorar mis habilidades en HTML y CSS. Aunque funcional, el portafolio cuenta con información limitada y está diseñado de manera muy simple.
