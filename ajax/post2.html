Segundo Proyecto - Digidex

Desarrollé una enciclopedia básica de Digimons utilizando HTML, CSS, JavaScript y Bootstrap. La aplicación consume una API pública de Digimons para obtener la información a mostrar. En la interfaz se muestra el nombre del Digimon, una imagen y su nivel. Usé Bootstrap para dar formato y responsividad a la página. Este proyecto me permitió practicar el consumo de APIs REST, manejo del DOM y estilos con Bootstrap en una aplicación web sencilla. La aplicación muestra la información básica del Digimon seleccionado, pero no tiene otras funcionalidades.
