Cuarto Proyecto - Nimrodel Restaurant

Desarrollé un sitio web simple para el restaurante Nimrodel utilizando HTML, CSS y Bootstrap. El sitio cuenta con una página de inicio que muestra los platillos del menú, una sección sobre el restaurant y formularios para contacto y reservación. Usé Bootstrap para crear un diseño responsive. Los formularios de contacto y reserva son sólo mockups y no están conectados a ninguna funcionalidad backend, fueron implementados con propósitos de práctica de frontend. Este proyecto me permitió practicar y aplicar responsividad con Bootstrap, crear un diseño web básico para un negocio y maquetar formularios sencillos.
